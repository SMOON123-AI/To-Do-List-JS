# To-Do List App 📝

A dynamic and interactive To-Do List application built using **HTML, CSS, and JavaScript**.
This app allows users to manage daily tasks efficiently with features like adding, editing, deleting, and saving tasks.

## 🚀 Live Demo
https://smoon123-ai.github.io/To-Do-List-JS/

## ✨ Features

* Add new tasks with a due date
* Delete tasks
* Edit task name (Rename)
* Update task date (Redate)
* Reset entire task list
* Press **Enter** to quickly add tasks
* Data persistence using **localStorage**
* Responsive and interactive UI

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation & Event Handling)

## ⚙️ How It Works

* Tasks are stored as objects in an array
* The array is saved in the browser using **localStorage**
* The UI is dynamically updated whenever a task is added, edited, or deleted
